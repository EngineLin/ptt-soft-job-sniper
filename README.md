# pttjobsniper

> 自動爬取 PTT soft-job 的工作機會，自動寄信給會員的平台。

## 目前進度：
* 前端輸入信箱的頁面(OK) 未深入加強畫面效果
* 後端信箱驗證系統/API (OK)


## Prototyping core function

1. 爬蟲 + 資料判斷
2. 資料儲存到資料庫
3. 當資料更新主動丟資料給使用者
4. 使用者輸入信箱 -> 信箱判斷 (token) (checked)
